**PCA-Guided CNN nhận diện cảm xúc**
- Các bước cài đặt
  +	Tạo venv bằng Python 3.10: python3.10 -m venv venv
  +	Kích hoạt venv: venv\Scripts\activate
  +	Cài đặt thư viện: pip install -r requirements.txt
  +	Chạy localhot demo: streamlit run app.py
  +	Tải bộ dataset FER2013 tại: https://www.kaggle.com/datasets/msambare/fer2013
